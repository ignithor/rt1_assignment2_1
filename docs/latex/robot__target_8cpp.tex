Description\+:

This node allows the user to set a target position for the robot or cancel the current goal. It subscribes to the {\ttfamily /odom} topic to get the robot\textquotesingle{}s current position and publishes a simplified version of the odometry data to {\ttfamily /odom\+\_\+simplified}. It also provides an interactive command-\/line interface for setting or canceling goals.
